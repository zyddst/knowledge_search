# 新人指南

## 第一天

1. **领设备**: 到行政部张姐处领取笔记本电脑和显示器，需要签字确认。
2. **开账号**: IT 会帮你开通企业邮箱、企业微信、GitLab、Confluence 账号。默认密码会在企业微信发给你，首次登录需要修改。
3. **加群**: 加入部门企业微信群，团队日常沟通都在企业微信。

## 开发环境搭建

1. 安装 VS Code（插件：Python、ESLint、Prettier）
2. 安装 Git，配置 SSH key 添加到 GitLab
3. 克隆项目仓库：`git clone git@gitlab.company.com:team/main-project.git`
4. 安装 Docker Desktop，用于本地开发环境
5. Python 项目用 Poetry 管理依赖：`poetry install`
6. 前端项目用 pnpm：`pnpm install`

## 代码规范

- Python: 遵循 PEP 8，用 Black 格式化，行宽 100
- TypeScript: 用 ESLint + Prettier，禁止 `any` 类型
- 提交信息格式: `type(scope): description`，如 `feat(auth): add login page`
- 所有 PR 需要至少一个人 Review 通过才能合并
- 合并前必须通过 CI 的 lint + test 检查

## 报销流程

1. 保留所有发票，金额超过 200 元的需要在发票背面写明用途
2. 每月 25 号前提交报销申请，财务会在次月 5 号统一打款
3. 在 OA 系统 → 费用报销 → 新建报销单 → 上传发票照片 → 提交审批
4. 差旅报销需要额外附上出差申请单
